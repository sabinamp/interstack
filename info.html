Objectives

Interstack Connections. Which Stacks have common registered users?

  • Are there many users of the "Mathematics Stack" maybe also frequently active on the "Ask Different Stack"?
  • Are the users active on the Stackoverflow Stack also active on the Web Applications Stack?
  • Are the users active on the Home Improvement Stack also active on the Photography Stack?
  • Besides the expected strong connections between:
    • Arqade and Arqade Meta,
    • Home Improvement and Home Improvement Meta,
    • Mathematics and Mathematics Meta
    • Super User and Meta Super User
    what other strong connections between stacks are there? Which stacks seem to share a great number of users?

Metrics and indicators that show relevant issues in our analysis

Degree Centrality

Partitioning by degree - Node size and color have been applied based on the degree of a node.

Edge Color

In order to highlight weighted edges, we gave them a green color that make the stronger edges more visible in the final display.

Layout
The circular layout visualisation shows nodes sorted based on their degree centrality value anticlockwise.
The Fruchterman Reingold layout emphasizes the strong connectivity between the nodes with the highest degree centrality:
Ask Different - Mathematics - Mathematics Meta - Ask Ubuntu - Arqade - Web Applications - Arqade Meta
